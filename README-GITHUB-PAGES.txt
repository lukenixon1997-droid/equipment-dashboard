# Parts Bible GitHub Pages Upload

This folder is ready for GitHub Pages.

## Upload steps

1. Unzip `parts-bible-github-ready.zip`
2. Open the unzipped folder
3. Upload everything inside this folder to your GitHub repository
4. Make sure `index.html` is at the top level of the repo
5. In GitHub, go to Settings > Pages
6. Source: Deploy from a branch
7. Branch: main
8. Folder: / root
9. Save

Your test site will publish as a GitHub Pages link.
